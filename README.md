# 欢迎使用YFramework渣渣框架


## 主要功能

YFramework是使用kotlin&Java混合编写的，主要包括以下功能：

 1. 具有**MVC、MVP、MVVM**架构
 2. 支持Aop切面编程：**Aspect**
 3. 封装**权限、拍照、图库**访问
 4. 集成第三方**分享、支付、登录**功能
 5. 注解式**DB**、并支持**协程，子线程**访问
 6. 使用主流的**RxJava2、retrofit2、RxBus**
 7. 封装view圆角变色 **selector库**
 8. 封装**sp**，使用内存缓存，子线程存储sp
